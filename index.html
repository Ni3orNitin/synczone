FIREBASE SETUP NEEDED

This app needs Firebase for real-time rooms.
It takes about 5 minutes to set up for free.

01Go to console.firebase.google.com → Create a project (free)
02Click Add app → Web → copy the firebaseConfig object
03Go to Realtime Database → Create database → Start in test mode
04Open config.js in your project and paste your config values
05Save and redeploy to Vercel → Done!
